## Learned
- MHeironimus ArduinoJoystickLibrary presents the controller as USB HID directly — no serial data is ever sent, which is why `screen` and serial listeners showed nothing
- Electron 20+ enables `sandbox: true` by default, which breaks ESM `import` in preload scripts — fix is `.cjs` extension to force CommonJS
- `distutils` was removed in Python 3.12+; `pip install setuptools` restores it for `node-gyp` builds

## Friction
- Multiple layered Electron issues (env var, preload format, native module ABI) required step-by-step debugging before the serial/HID root cause was found
- Web Gamepad API requires at least one button press to activate the gamepad in the browser before polling returns data
